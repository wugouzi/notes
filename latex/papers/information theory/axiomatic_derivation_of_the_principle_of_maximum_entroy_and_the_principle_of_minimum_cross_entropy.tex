\subsection{The Maximum Entropy Principle and the Minimum Cross-Entropy Principle}
\label{sec:org3ea0e5e}
Suppose you know that a system has a set of possible states \(x_i\) with unknown probabilities
\(q^\dag(x_i)\), and you then learn \emph{constraints} on the distribution \(q^\dag\): either values of
certain expectations \(\sum_iq^\dag(x_i)f_k(x_i)\) or bounds on these values. Suppose you need to
choose a distribution \(q\) that is in some sense the best estimate of \(q^\dag\) given what you know.
Usually there remains an infinite set of distributions that are not ruled out by the constraints.
Which one should you choose?

The principle of maximum entropy states that, of all the distributions \(q\) that satisfy the
constraints, you should choose the one with the largest entropy \(-\sum_iq(x_i)\log(q(x_i))\)

The principle of minimum cross-entropy is a generalization that applies in cases when a prior
distribution \(p\) that estimates \(q^\dag\) is known in addition to the constraints. The principle
states that, of the distributions \(q\) that satisfy the constraints, you should choose the one with
the least cross-entropy \(\sum_iq(x_i)\log(q(x_i)/p(x_i))\). Minimizing cross-entropy is equivalent to
maximizing entropy when the prior is a uniform distribution. Unlike entropy maximization,
cross-entropy minimization generalizes correctly for continuous probability densities. One then
minimizes the functional
\begin{equation}
\label{1}
H(q,p)=\int q(x)\log(q(x)/p(x))dx
\end{equation}
\subsection{Justifying the Principles as General Methods of Inference}
\label{sec:org09f8161}
We prove that the maximum entropy principle is correct in the following sense: maximizing any function
but entropy will lead to inconsistencies unless that function and entropy have identical maxima.

Stated differently, we prove that, given new constraint information, there is only one distribution
satisfying these constraints that can be chosen by a procedure that satisfies the consistency axioms;
this unique distribution can be obtained by maximizing entropy.

Informally, our axioms may be phrased as follows:
\begin{enumerate}
\item \textbf{Uniqueness}: The result should be unique
\item \textbf{Invariance}
\item \textbf{System Independence}
\item \textbf{Subset Independence}
\end{enumerate}
\section{Definitions and Notation}
\label{sec:org4ab3d98}
We use lowercase boldface roman letters for system states, which may be multidimensional, and
uppercase boldface roman letters for sets of system states. Let \(\bx\) be a state of some system that
has a set \(\bD\) of possible states, let \(\cald\) be the set of all probability densities \(q\) on
\(\bD\) s.t. \(q(\bx)\ge 0\) for \(\bx\in\bD\) and
\begin{equation*}
\int_{\bD}q(\bx)d\bx=1
\end{equation*}
We use a superscript dagger to distinguish the system's unknown ``true'' state probability density
\(q^\dag\in\cald\).

New information takes the form of linear \textbf{equality constraints}
\begin{equation*}
\int_{\bD}q^\dag(\bx)a_k(\bx)d\bx=0
\end{equation*}
and \textbf{inequality constraints}
\begin{equation*}
\int_{\bD}q^\dag(\bx)c_k(\bx)d\bx\ge 0
\end{equation*}
for known sets of bounded functions \(a_k\) and \(c_k\). The probability densities that satisfy such
constraints always comprise a closed convex subset of \(\cald\). (A set \(\calj\subseteq\cald\) is
\textbf{convex} if, given \(0\le A\le 1\) and \(q,r\in\calj\), it contains the weighted average \(Aq+(1-A)r\))

Furthermore, any closed convex subset of \(\cald\) can be defined by equality and inequality
constraints. We express constraints in these terms, using the notation \(I=(q^\dag\in\calj)\), to mean
that \(q^\dag\) is a member of the closed convex set \(\calj\subseteq\cald\). We refer to \(I\) as a
\textbf{constraint} and to \(\calj\) as a \textbf{constraint set}.

Let \(p\in\cald\) be some \emph{prior} density that is an estimate of \(q^\dag\) obtained, by any means,
prior to learning \(I\). We require that priors be strictly positive
\begin{equation}
\label{5}
p(\bx\in\bD)>0
\end{equation}
Given a prior \(p\) and new information \(I\), the \textbf{posterior} density \(q\in\calj\) that results from
taking \(I\) into account is chosen by minimizing a functional \(H(q,p)\) in the constraint set
\(\calj\):
\begin{equation}
\label{6}
H(q,p)=\min_{q'\in\calj}H(q',p)
\end{equation}

We introduce an ``information operator'' \(\circ\) that expresses \eqref{6} using the notation
\begin{equation*}
q=p\circ I
\end{equation*}

For some other functional \(F(q,p)\), suppose \(q\) satisfies \eqref{6} iff it satisfies
\begin{equation*}
F(q,p)=\min_{q'\in\calj}F(q',p)
\end{equation*}
Then we say that \(F\) and \(H\) are \textbf{equivalent}. If \(F\) and \(H\) are equivalent, the operator
\(\circ\) can be realized using either functional.

If \(H\) has the form \eqref{1}, then \eqref{7} expresses the principle of minimum cross-entropy.

In making the restriction \eqref{5} we assume that \(\bD\) is the set of states that are possible
according to prior information. We do not impose a similar restriction on the posterior \(q=p\circ I\)
since \(I\) may rule out states currently thought to be possible. If this happens, then \(\bD\) must
be redefined before \(q\) is used as a prior in a further application of \(\circ\). For simplicity, we
impose on the information \(I\) the restriction that there exists at least one density \(q\in\calj\)
with \(H(q,p)<\infty\)

For some subset \(\bS\subseteq\bD\) of states and \(\bx\in\bS\), let
\begin{equation*}
q(\bx\mid\bx\in\bS)=\frac{q(\bx)}{\int_{\bS}q(\bx')d\bx'}
\end{equation*}
be the \textbf{conditional density}, given \(\bx\in\bS\), corresponding to any \(q\in\cald\). We use
\begin{equation*}
q(\bx\mid\bx\in\bS)=q*\bS
\end{equation*}
as a shorthand.

When \(\bD\) is a discrete set of system states, densities are replaced by discrete distributions and
integrals by sums in the usual way. We use lowercase boldface roman letters for discrete probability
distributions, which we consider to be vectors; for example, \(\bq=q_1,\dots,q_n\).
\section{The Axioms}
\label{sec:orgccec751}
We assume, throughout, a system with possible states \(\bD\) and probability densitiy
\(q^\dag\in\cald\).

\begin{axiom}[Uniqueness]
The posterior \(q=p\circ I\) is unique for any prior \(p\in\cald\) and new information
\(I=(q^\dag\in\calj)\), where \(\calj\subseteq\cald\)
\end{axiom}

Let \(\Gamma\) be a coordinate transformation from \(\bx\in\bD\) to \(\by\in\bD'\) with
\((\Gamma q)(\by)=\bJ^{-1}q(\bx)\) where \(\bJ\) is the Jacobian \(\bJ=\abs{\partial(\by)/\partial(\bx)}\).

Let \(\Gamma\cald\) be the set of densities \(\Gamma q\) corresponding to densities \(q\in\cald\). Let
\((\Gamma\calj)\subseteq(\Gamma\cald)\) correspond to \(\calj\subseteq\cald\). Then, for any prior
\(p\in\cald\) and new information \(I=(q^\dag\in\calj)\),
\begin{equation*}
(\Gamma p)\circ(\Gamma I)=\Gamma(p\circ I)
\end{equation*}
holds, where \(\Gamma I=((\Gamma q^\dag)\in(\Gamma\calj))\)
\end{axiom}

Justification: We expect the same answer when we solve the same problem in two different coordinate systems

Suppose there are two systems, with sets \(\bD_1\), \(\bD_2\) of states and probability densities of
states \(q_1^\dag\in\cald_1\), \(q_2^\dag\in\cald_2\). Then we require the following axiom.
\begin{axiom}[System Independence]
Let \(p_1\in\cald_1\) and \(p_2\in\cald_2\) be prior densities. Let \(I_1=(q_1^\dag\in\calj_1)\) and
\(I_2=(q_2^\dag\in\calj_2)\) be new information about the two systems, where
\(\calj_1\subseteq\cald_1\) and \(\calj_2\subseteq\cald_2\). Then
\begin{equation*}
(p_1p_2)\circ(I_1\wedge I_2)=(p_1\circ I_1)(p_2\circ I_2)
\end{equation*}
holds
\end{axiom}

Justification: Instead of \(q_1^\dag\) and \(q_2^\dag\) we could describe the systems using the joint
density \(q^\dag\in\cald_{12}\). If the two systems were independent, then the joint density would
satisfy
\begin{equation}
\label{12}
q^\dag(\bx_1,\bx_2)=q_1^\dag(\bx_1)q_2^\dag(\bx_2)
\end{equation}
Now the new information about each system can also be expressed completely in terms of the joint
density \(q^\dag\). For example, \(I_1\) can be expressed as \(I_1=(q^\dag\in\cali_1')\), where
\(\cali_1\subseteq\cald_{12}\) is the set of joint densities \(q\in\cald_{12}\) s.t.
\(q_1\in\cali_1\), where
\begin{equation*}
q_1(\bx_1)=\int_{\bD_2}q(\bx_1,\bx_2)d\bx_2
\end{equation*}
\(I_2\) can be expressed similarly. Now, since two priors together define a joint prior \(p=p_1p_2\),
it follows that there are two ways to take the new information \(I_1\) and \(I_2\) into account: we
can obtain separate posteriors \(q_1=p_1\circ I_1\) and \(q_2=p_2\circ I_2\), or we can obtain a joint
posterior \(q=p\circ(I_1\wedge I_2)\). Because \(p_1\) and \(p_2\) are independent, and because
\(I_1\) and \(I_2\) give no information about any interaction between the two systems, we expect these
two ways to be related by \(q=q_1q_2\), whether or not \eqref{12} holds

\begin{axiom}[Subset Independence]
Let \(\bS_1,\dots,\bS_n\) be disjoint sets whose union is \(\bD\), and let \(p\in\cald\) be any known
prior. For each subset \(\bS_i\), let \(I_i=(q^\dag*\bS_i\in\cali_i)\) be new information about the
conditional density \(q^\dag*\bS_i\), where \(\cali_i\subseteq\cals_i\) is the set of densities on
\(\bS_i\). Let \(M=(q^\dag\in\calm)\) be new information giving the probability of being in each of
the \(n\) subsets, where \(\calm\) is the set of densities \(q\) that satisfy
\begin{equation}
\label{13}
\int_{\bS_i}q(\bx)d\bx=m_i
\end{equation}
for each subset \(\bS_i\), where the \(m_i\) are known values. Then
\begin{equation*}
(p\circ(I\wedge M))*\bS_i=(p*\bS_i)\circ I_i
\end{equation*}
holds, where \(I=I_1\wedge I_2\wedge\dots\wedge I_n\)
\end{axiom}

Justification: This axiom concerns situations in which the set of states \(\bD\) decomposes naturally
into disjoint subsets \(\bS_i\), and new information \(I_i\) is obtained about the conditional
probability densities \(q^\dag*\bS_i\) in each subset. One way of accouting for this information is to
obtain a conditional posterior \(q_i=(p*\bS_i)\circ I_i\) from each conditional prior \(p*\bS_i\).
Another way is to obtain a posterior \(q=p\circ I\) for the whole system, where
\(I=I_1\wedge\dots\wedge I_n\). Two results should be related by \(q*\bS_i=q_i\) or
\begin{equation}
\label{15}
(p\circ I)*\bS_i=(p*\bS_i)\circ I_i
\end{equation}
Moreover, suppose that we also learn the probability of being in each of the \(n\) susbets. That is,
we learn \(M=(q^\dag\in\calm)\) where \(\calm\) is the set of densities \(q\) that satisfy \eqref{13}
for each subset \(\bS_i\). The known numbers \(m_i\) are the probabilities that the system is in a
state within \(\bS_i\). The \(m_i\) satisfy \(\sum_im_i=1\). Taking \(M\) into account should not
affect the conditional densities that result from taking \(I\) into account. We therefore expect a
more general version of \eqref{15} to hold, namely \eqref{14}
\section{Consequences of the Axioms}
\label{sec:orga7de477}
\subsection{Summary}
\label{sec:org59b65e4}
We establish our main result in four steps.
\begin{enumerate}
\item The subset independence axiom and a special case of the invariance axiom together restrict \(H\) to
functionals that are equivalent to the form
\begin{equation*}
F(q,p)=\int_{\bD}f(q(\bx),p(\bx))d\bx
\end{equation*}
for some function \(f\). We call this the ``sum form''
\item The general case of the invariance axiom restricts \(H\) to functionals that are equivalent to the
form
\begin{equation*}
F(q,p)=\int_{\bD}q(\bx)h(q(\bx)/p(\bx))d\bx
\end{equation*}
for some function \(h\).
\item Using independence axioms, we show that if \(H\) is a functional that satisfies all four axioms,
then \(H\) is equivalent to cross-entropy.
\item Cross-entropy satisfies the axioms
\end{enumerate}
